Intro to decorators

2023-05-04 ::
#python  #decorators 
A decorator is a special pattern where a callable takes a single parameter which is another callable, mainly classes and functions and return something, usually another callable. This can happen either on the decorated definition or later on. In python the at (@) operator is used as syntactic sugar.
